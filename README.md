# Meridian Credit Union — Engineering Standards

The standard of record for how Meridian builds software.

- **[Engineering handbook](handbook/engineering-handbook.md)** — values, definition of done,
  coding standards, API guidelines, security baseline, change management. Clauses are
  numbered; cite them by number (e.g. "per §4.4") in ADRs, PRs, and reviews.
- **[Architecture decision records](adr/)** — one file per decision, template included.

Owned jointly by the Member Experience and Staff Tools squads. Changes land by pull request
with owning-squad review, like any other repo.
